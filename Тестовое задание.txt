2. Java, C# -- высокоуровневые языки общего назначения. Используются для написания web и десктопных приложений.
С++ -- более низкоуровневый язык, его используют, когда важна скорость и эффективность работы программ. 
Swift -- для программ под систему IOS.
3. Компилятор переводит код программы с высокоуровневого языка на язык машинных команд, понятных процессору, а затем исполняет его.
Также существуют интерпретируемые языки программирования, вроде Javascript. В них интерпретатор непосредственно выполняет (интерпретирует) текст программы без предварительного перевода.
4. Фреймворк -- это набор библиотек, которые облегчают разработку программ. Фреймворк -- это готовое решение, которое увеличивает скорость написания кода, но ограничивает нас в гибкости. 
Примеры: Bootstrap, Spring, Vue.js, Ext.js, ASP.NET.
5. http -- протокол передачи данных, используемый для получения данных с веб-сайтов. Https -- расширение протокола HTTP, поддерживающее шифрование по протоколам SSL и TLS.
